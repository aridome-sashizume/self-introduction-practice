# 自己紹介ページの作成
HTMLを用いて自己紹介ページを作成
自己紹介ページには自身の趣味等を組込み、問い合わせを受けるためのフォームを作成

#フォームの作成方法
## 順序付きリストの作成
'<&ol>':Order Listでタグ付けし、各項目を<li>で設定

## 順序なしリストの作成
'<ul>':Unorder Listでタグ付けし、各項目を<li>で設定

## 画像の入力（'<img>'タグ）
'<img>'で参照項目をsrcをくくり、画像をページに組み込む
altで代替テキストを設定

##　リンクの組込み
'<a href="URL">'でアンカーを設定
target="blank"とすることで新規ページを開くように設定

## フォームの作成
'<form>'で入力フォームでまとめて入力する
各入力項目は以下のようにまとめる

### ユーザに入力させる項目（'<iput>'タグ）
ユーザに入力させるためのテキストボックスを作成

### ユーザに選択させる項目（'<select>'タグ）
各項目を'<option>'タグで設定し、ドロップダウンで選択

### ユーザに入力させる項目（'<textarea>'タグ）
複数行を入力させるためのテキストエリアを作成
行数については'<row = "行数">'で指定する
