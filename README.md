# 自己紹介ページの作成
HTMLを用いて自己紹介ページを作成
自己紹介ページには自身の趣味等を組込み、問い合わせを受けるためのフォームを作成

#フォームの作成方法
## 順序付きリストの作成
&lt;ol&gt;:Order Listでタグ付けし、各項目を<li>で設定


## 順序なしリストの作成
&lt;ul&gt;':Unorder Listでタグ付けし、各項目を<li>で設定

## 画像の入力（'<img>'タグ）
&lt;img&gt;で参照項目をsrcをくくり、画像をページに組み込む
altで代替テキストを設定

##　リンクの組込み
&lt;a href="URL"&gt;でアンカーを設定
target="blank"とすることで新規ページを開くように設定

## フォームの作成
&lt;form&gt;で入力フォームでまとめて入力する
各入力項目は以下のようにまとめる

### ユーザに入力させる項目（&lt;iput&gt;タグ）
ユーザに入力させるためのテキストボックスを作成

### ユーザに選択させる項目（&lt;select&gt;タグ）
各項目を&lt;option>'タグで設定し、ドロップダウンで選択

### ユーザに入力させる項目（&lt;textarea&gt;タグ）
複数行を入力させるためのテキストエリアを作成
行数については&lt;row = "行数"&gt;で指定する
